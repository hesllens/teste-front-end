#entrar na pasta quasar-project
via vscode

## instalar dependencias
```bash
yarn
# ou
npm install
```
### (opcional) instalar global cli quasar
yarn global add @quasar/cli
# ou
npm install -g @quasar/cli

#### startar o app)
```bash
npx quasar dev
# ou
quasar dev (caso tenha global cli)
# ou
yarn quasar dev

```
